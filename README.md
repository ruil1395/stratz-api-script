# Скрипт для работы с API Stratz

Этот проект содержит простой Python-скрипт для получения детальной информации о матчах Dota 2 с помощью официального GraphQL API сайта stratz.com.

## Возможности

-   Получение информации о победителе, длительности и счете матча.
-   Вывод составов обеих команд (игроки и герои).
-   Отображение индивидуальной статистики игроков (K/D/A, GPM).
-   Простая настройка и использование.

## Настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ВАШЕ_ИМЯ/stratz-api-script.git
    cd stratz-api-script
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Добавьте ваш API-ключ:**
    Откройте файл `stratz_script.py` и вставьте ваш API-ключ от Stratz в переменную `API_KEY`.

## Использование

Скрипт запускается из командной строки. В качестве аргумента необходимо передать ID матча.

**Пример:**
```bash
